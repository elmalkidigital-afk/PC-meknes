## 🚀 ACTIONS IMMÉDIAT (AUJOURD'HUI/DEMAIN)

### ÉTAPE 1: Vérifier la mise en ligne ⏱️ 2 MIN

```bash
# Dans le navigateur, vérifier:
https://pc-meknes.fr                    # Doit charger
https://pc-meknes.fr/sitemap.xml        # Doit afficher du XML
https://pc-meknes.fr/robots.txt         # Doit afficher du texte
```

### ÉTAPE 2: Ajouter meta tag Google Search Console ⏱️ 3 MIN

1. Aller sur: https://search.google.com/search-console
2. Cliquer: **"Ajouter une propriété"**
3. Saisir: `pc-meknes.fr`
4. **Copier le meta tag** fourni (ressemble à ça):
```html
<meta name="google-site-verification" content="1a2b3c4d5e6f7g8h9i0j" />
```
5. Ouvrir: `/public/index.html`
6. Chercher: `<link rel="canonical" href="https://pc-meknes.fr">`
7. **Ajouter juste après:**
```html
<!-- Google Search Console Verification -->
<meta name="google-site-verification" content="1a2b3c4d5e6f7g8h9i0j" />
```
8. Sauvegarder et **déployer**
9. Retour à Google → Cliquer **"Vérifier"**

✅ **Vous devriez voir:** "Propriété vérifiée"

### ÉTAPE 3: Soumettre le sitemap ⏱️ 2 MIN

1. Dans Google Search Console
2. Aller à: **"Sitemaps"** (menu gauche)
3. Cliquer: **"Nouvelle soumission"**
4. Saisir: `https://pc-meknes.fr/sitemap.xml`
5. Cliquer: **"Soumettre"**

✅ **Vous devriez voir:** "Sitemap reçu"

### ÉTAPE 4: Demander l'indexation manuelle ⏱️ 1 MIN

1. Dans Google Search Console
2. Barre de recherche **en haut**
3. Saisir: `https://pc-meknes.fr`
4. Cliquer: **"Demander l'indexation"**

✅ **Vous devriez voir:** "Demande de réindexation envoyée"

### ÉTAPE 5: Tester avec Google PageSpeed ⏱️ 2 MIN

1. Aller sur: https://pagespeed.web.dev/
2. Saisir: `https://pc-meknes.fr`
3. Vérifier les scores (viser 90+)
4. Noter les points à améliorer

### ÉTAPE 6: Tester le schema ⏱️ 1 MIN

1. Aller sur: https://search.google.com/test/rich-results
2. Saisir: `https://pc-meknes.fr`
3. Vérifier qu'il y a des éléments structurés
