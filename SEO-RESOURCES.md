## 📊 MÉTRIQUES CLÉS À SURVEILLER

### Google Search Console
- **Clics:** Nombre de visites depuis Google Search
- **Impressions:** Fois que le site apparaît dans les résultats
- **CTR:** Click-Through Rate (clics ÷ impressions)
- **Position:** Classement moyen

**Accès:** https://search.google.com/search-console

### Google Analytics 4
- **Sessions:** Nombre de visites
- **Utilisateurs:** Nombre de visiteurs uniques
- **Taux de rebond:** % de visites sans interaction
- **Temps sur site:** Engagement moyen
- **Conversions:** Contacts/leads générés

**Accès:** https://analytics.google.com

### PageSpeed Insights
- **LCP (Largest Contentful Paint):** < 2.5s (bon)
- **FID (First Input Delay):** < 100ms (bon)
- **CLS (Cumulative Layout Shift):** < 0.1 (bon)

**Accès:** https://pagespeed.web.dev/
